### r-cran-ggplot2
Installing r-cran-ggplot2 package on Ubuntu 16.04 (Xenial Xerus) is as easy as running the following command on terminal:

<pre>
sudo apt-get update
sudo apt-get install r-cran-ggplot2
</pre>
